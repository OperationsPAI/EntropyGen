# Learn Command

Analyze current session and extract patterns and experiences worth saving.

## What to Extract

1. **Problem-solving patterns** — Non-obvious debugging paths
2. **Architectural decisions** — What choices were made and why
3. **Project conventions** — Agreements confirmed during discussion
4. **Design patterns** — Recurring design approaches

## Workflow

1. Review key decisions and solutions from the session
2. Identify insights with highest reuse value
3. Draft skill file
4. Save after user confirmation

## Output

Save to project memory or appropriate location in `.claude/`.

## What NOT to Extract

- Simple typo fixes
- One-off issues
- Content already recorded in CLAUDE.md
